#### Зміни в промпті:

**Файл:** `server/constants/prompts.ts`

### 2. 🔮 Інтерактивні підказки (Hints)

Після кожного обчислення AI автоматично генерує 2-3 питання про елементи у виразі.

#### Як це працює:

1. **User вводить:** `123+567*sqrt(144) - log(10)`
2. **AI генерує hints:**
   - "Що таке квадратний корінь?"
   - "Навіщо логарифм?"
   - "Як працює sqrt?"

3. **User клікає** на hint → отримує коротку та зрозумілу відповідь!

#### Приклад відповіді:
> **Що таке логарифм?**  
> Логарифм показує, в який степінь треба піднести число, щоб отримати інше. Корисний для роботи з великими числами! 🎯

## 📁 Нові файли

### API Endpoints:

1. **`/server/api/hints.post.ts`**
   - Генерує питання на основі виразу
   - Використовує `HINT_GENERATOR_PROMPT`
   - Повертає масив 2-3 питань

2. **`/server/api/hint-answer.post.ts`**
   - Відповідає на питання коротко та просто
   - Використовує `HINT_ANSWER_PROMPT`
   - Max 2 речення

### Prompts:

3. **`MATH_TEACHER_PROMPT`** (оновлено)
   - Більш дружній та живий
   - Дозволені емодзі та емоції

4. **`HINT_GENERATOR_PROMPT`** (новий)
   - Генерує релевантні питання
   - JSON формат відповіді

5. **`HINT_ANSWER_PROMPT`** (новий)
   - Короткі пояснення
   - Практичний підхід

## 🧪 Тестування

### Спробуйте різні вирази:

1. **Тригонометрія:**
   ```
   sin(pi/2) + cos(0)
   ```

2. **Корені та степені:**
   ```
   sqrt(16) + 2^3
   ```
   Hints: "Як працює sqrt?", "Що означає степінь?"

3. **Логарифми:**
   ```
   log(100) + ln(e)
   ```
   Hints: "Що таке логарифм?", "Навіщо ln?"

4. **Константи:**
   ```
   pi * e + phi
   ```
   Hints: "Що означає Pi?", "Хто такий Phi?"
